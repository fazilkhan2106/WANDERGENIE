# WanderGenie – AI Travel Planner

## 📌 Overview

**WanderGenie** is a smart travel planning application designed to simplify the process of organizing trips. Planning a journey often requires switching between multiple websites, managing dates, remembering destinations, and keeping track of travel details. This process can be time-consuming and confusing.

WanderGenie solves this problem by providing a **personalized travel planning platform** where users can easily create, store, and manage their travel plans in one place.

Instead of only suggesting destinations, WanderGenie allows users to actively design their own trips by selecting a destination, choosing travel dates, and saving the plan. The application stores all trip details securely so that users can access or modify them anytime.

In simple terms, WanderGenie works like a **digital travel assistant and personal travel diary**, helping travelers focus more on their journey and less on organizing it.

# 🏗️ System Architecture

The application follows a **Full-Stack Architecture**:

```
Mobile App (React Native)
        │
        │ API Requests (Axios)
        ▼
Backend Server (Node.js + Express)
        │
        ▼
Database (MongoDB)
```

### Workflow

1. User enters trip details in the mobile application  
2. Data is sent to the backend using APIs  
3. The backend processes the request  
4. Trip data is stored securely in MongoDB  
5. Users can later retrieve or update their saved trips
